# yt-tags
Lookup the tags of a Youtube video.

# installation
- Download from https://github.com/xyzpw/yt-tags/archive/refs/heads/main.zip
- Unzip file
- Add execution perms to file (optional)
- Execute script
## execution (linux)
Downloading and executing example with Linux.
```bash
wget -O yt-tags.zip https://github.com/xyzpw/yt-tags/archive/refs/heads/main.zip
unzip yt-tags.zip
```
Optionally, you can add execution perms if you want to execute:
```bash
chmod +x yt-tags.py
./yt-tags.py -i dQw4w9WgXcQ # ;)
```
Tags are printed immediately after running script with the `-i` argument.
